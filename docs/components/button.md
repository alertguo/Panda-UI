## 说明

### icon 
1.目前有如下几种：`setting`、`download`、`thumbs-up`、`down`、`left`、`right`

2.另外`loading` 可为 `true` 或者 `false`；当为 true 的时候，替换掉`icon` 
### icon-position
默认值：`false` - 在按钮左边

设置为 `true` 在右边
